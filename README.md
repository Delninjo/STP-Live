# STP Live — gotov ZIP

## Najbrže pokretanje (lokalno)
1) Instaliraj Node.js (LTS).
2) U folderu projekta:

```bash
npm install
npm run dev
```

Otvori: http://localhost:3000

## Instalacija na mobitel (kao app)
Kad app bude online (Vercel) ili kad ga otvoriš na mobitelu:
- iPhone: Safari → Share → Add to Home Screen
- Android: Chrome → ⋮ → Install app / Add to Home screen

## Što ima unutra
- Kamera: otvara LiveCamCroatia (Sljeme vidikovac)
- Vrijeme: DHMZ Puntijarka (trenutno) + Open-Meteo (7 dana)
- Žičara: radno vrijeme + obavijesti (zicarasljeme.hr)
